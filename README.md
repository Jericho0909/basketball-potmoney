## Play4Pot Website:

Play4Pot is a web-based application built using React, TypeScript, and Tailwind CSS.

The website consists of two main pages: the Admin Page and the Voter Page.

The Admin Page is used to manage matches, where the admin sets up the two players who will compete, including the match location, start time, and the initial pot amount.

The Voter Page is designed for users, where they can view the current matchup, see the voting rules, and cast their vote for their chosen player.

## Features:

- Create and manage player matchups
- Set match location, start time, and initial pot amount
- View two competing players in a head-to-head format
- Users can vote for their chosen player
- Voting rules are displayed to guide users
- Displays the current total pot in a readable
- Backend support using JSON Server (with planned Firebase integration)
- Fully responsive design

## Tech Stack:

- Frontend: React, TypeScript, Tailwind CSS, Framer Motion

- Backend:  JSON Server / Firebase

- Hosting:  Netlify

- Version Control:  Git + GitHub


## Future Improvements:

- Persist admin input data so player and match information will not be lost on page refresh

- Display player scores in real time during voting
